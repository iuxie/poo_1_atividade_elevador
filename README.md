# POO 1 - Atividade Elevador

## Descrição

Este projeto foi desenvolvido como parte da disciplina de **Programação Orientada a Objetos (POO)** e tem como objetivo representar o funcionamento de um elevador por meio de uma classe em Java.

A aplicação simula as principais operações de um elevador, respeitando regras de negócio e restrições definidas na atividade.

---

## Funcionalidades

O elevador implementa as seguintes ações:

*  Subir para um andar
*  Descer para um andar
*  Abrir a porta
*  Fechar a porta
*  Verificar se a porta está aberta ou fechada
*  Mostrar o andar atual

---

##  Restrições implementadas

O sistema respeita as seguintes regras:

* O elevador **não pode subir acima do número máximo de andares**
* O elevador **não pode descer abaixo do andar mínimo (0)**
* O elevador **não pode se mover com a porta aberta**
* O elevador **não pode subir para um andar abaixo**
* O elevador **não pode descer para um andar acima**

---

## Conceitos aplicados

* Programação Orientada a Objetos (POO)
* Encapsulamento
* Controle de estado com atributos privados
* Validação de regras de negócio
* Métodos públicos para interação com o objeto
